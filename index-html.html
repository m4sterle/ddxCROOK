ddxCROOK: A Clinical Reasoning Adventure

Welcome to ddxCROOK! This Python terminal game challenges your clinical reasoning skills as you navigate the intimidating world of hospital rotations with the tough but brilliant Dr. Crook.

Game Example:

Dr. Crook: "So, given these vital signs..."

What's your next move?

  1. 'Could we get more history about the headaches?'
  2. *Frantically google 'high BP + tachycardia' on your phone*
  3. 'RAPID RESPONSE!' *Reaches for the emergency button*
  4. 'Well, when we consider the sympathetic nervous system...'

Your choice: 1

Dr. Crook: "Ah, finally asking the right questions!" Dr. Crook's eyes light up.

Dr. Crook: "Patient reports episodic symptoms including headache, palpitations, and diaphoresis..."

This game features:

  • A medical diagnostic adventure with multiple decision points
  • Player stats tracking your anxiety, reputation, and clinical accuracy
  • A challenging case of pheochromocytoma to diagnose
  • The terrifying yet educational presence of Dr. Crook

Try It Yourself!

This is a terminal-based Python game. To play it:

  1. Clone the repository: git clone https://github.com/YourUsername/ddxCROOK.git
  2. Run the game: python terminal_game.py
